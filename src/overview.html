Overview

Implementation!!!

This software is implemented as a Private Permissioned Blockchain by using the Ethereum platform. We are using a Geth Client which is a Go implementation of the ethereum node. Using this Geth Client we can configure the number of nodes we need for the blockchain solution manually.

Smart contracts are written by using a Solidity a javascript programming language. This smart contracts will act as the back end for this particular project and these contracts are also got deployed in the blockchain before using it. These smart contracts are executed as per the requirement when a user feeds some information.


Each and every node of the network will have a updated blockchain ledger.This blockchain ledger will consists of the history of the block added to a blockchain ledger and thier related transaction history details. It includes the block hash,previous hash,difficulty level etc..


KYC documents are added to the blockchain by using the distributed file storage enivronment called "InterPlanary File Storage-IPFS.IPFS returns a hash and this hash is stored in blockchain and file is stored in IPFS


Client interface is developed by using the convetional web development tools like HTML,CSS,Javascript and a Bootstrap.It consists of the number of client forms for feeding the user data and thier related documents.
